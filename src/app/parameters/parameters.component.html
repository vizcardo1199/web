The Vibration Alarm Problem



Existing Vibration Alarm Standards only deal with “Overall Vibration”.

Back in the days it was fine, but then, there was no spectra, only overall.

With the usefulness of the vibration spectrum, overall alarms are no longer valid. Spectra have thousands of points. The overall can’t cover all of them. Old standards are no longer valid.



What are Spectral Bands?

Bands are pieces of the spectrum that contain several frequencies, usually dealing with specific vibration issues such as:
1x : Unbalance, looseness
2-3x: Misalignment
etc, etc. All vibration issues are covered.
Now, it makes sense to alarm on each spectral band.
But there is another problem: There are no standards for this, because machines are so different from each other, we would have to create alarm standards for each machine. That is a lot of standards!!



The PDM Monitor Solution:

The PDM Monitor did just that!
The PDM Monitor creates alarm standards for each of the bands for each type of machine. This allows the analyst to sort through the machines faster and concentrate on the machines with high alarms.
This is possible with a new concept in Vibration Monitoring, called the “Typical Machine”



The PDM Monitor Solution: Typical Machines

A typical machine is an ideal representative of a group of similar machines. Ideal in the sense that it has the best vibrations of such group.
The vibration of a typical machine is used as a baseline to establish alarms for all other machines belonging to its group.
But…. What are the actual alarms for a machine?
The analyst needs not to worry. This is the task of the PDM Monitor. The PDM Monitor will calculate the typical machine bands and establish alarms using its smart algorithms.



Alarm Levels

The PDM Monitor will establish alarms for each machine in the system based in the typical machine associated to it.
But what is an alarm level based on? It’s based on the user’s perception of the criticality of a machine. The most critical machines have the lowest alarm levels, the less critical, the highest alarm levels.
The PDM Monitor allows the users to set that for each machine, using three levels:
Smooth (Highest criticality machines)
Regular (Medium criticality machines)
Rough (Lowest criticality machines)



Alarm Communications

OK. What happens if a machine reaches the RED alarm level?
The PDM Monitor catches the alarm and if determined to have happened suddenly and resulted in a high vibration reading, it will send the users SMS text messages, e-mails, or any social media platform the user prefers.
The user can then go in the PDM Monitor and use its analysis tools to determine the problem.



Alarm Prediction

Using its advanced algorithms, the PDM Monitor will also predict the number of days a machine is projected to take to reach the RED alarm level.
