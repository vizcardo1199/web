PDM Monitor Sensors

The PDM Monitor can use any market sensor for vibration and temperature.
Using the power of MEMS, we have developed a multiple technology sensor which is a tri-axial accelerometer and a temperature sensor, all in one at an affordable cost.





How is the PDM Monitor different from existing vibration platforms?

The PDM Monitor is an Industry 4.0 System. Remote Cloud monitoring is the next step to route-based data monitoring.
Data is available worldwide as opposed to only in one local computer or network.
No more software purchasing or maintenance fees. Software is in the cloud or in a downloadable app. Updates happen over-the-air.
Expandability of the system is limitless.
Analysts can analyze data from anywhere in the world at any time.
